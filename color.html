色碼

一般來說，色碼有三種宣告方式
  • 名稱式: 像是 red, green, blue, pink ...等，可閱 這邊
  • RGB式: 給予R(紅)、G(綠)、B(藍)來決定顏色 每個數值範圍從 0 到 255
    例如：
    • 黑色
    • 白色
    • 紅色
    • 咖啡色
    • 深咖啡色
    • 綠色
    • 藍色
    • 黃色
    • 青色
  • 16進位式: 使用 #rrggbb 格式代表其色，# 為固定碼 rr與gg與bb分別代表：紅、綠、藍的16進位值。 每個數值從 00 到 ff
    • #000000 = 黑色
    • #ffffff = 白色
    • #ff0000 = 紅色
    • #960000 = 咖啡色
    • #320000 = 深咖啡色
    • #00ff00 = 綠色
    • #0000ff = 藍色
    • #ffff00 = 黃色
    • #00ffff = 青色
    • #bg-orange = 橘色
    • 紫色
    • 粉紅色
    • 灰色
