DOM 是 Document Object Model (文件物件模型) 的簡稱
故名思義，是以 操作文件裡的東西為目標 的物件。

document.getElementById 使用id抓取某個DOM (html標籤)

使用id抓取某個DOM，只抓到 1 或 null
  • document.getElementById(id名) ← 抓取
  • document.getElementById(id名).innerHTML ← 抓取裡面的內容
  • document.getElementById(id名).innerHTML = abc ← 取代裡面的內容

document.getElementsByTagName 使用tagName抓取某些DOM (html標籤)

使用tagName抓取某些DOM，抓到HTMLCollection物件，可使用陣列方法操作。
  • document.getElementsByTagName(tagName名) ← 抓取HTMLCollection
  • document.getElementsByTagName(tagName名)[0] ← 抓取內容裡的第1筆內容
  • document.getElementsByTagName(tagName名)[1] ← 抓取內容裡的第2筆內容
  • document.getElementsByTagName(tagName名)[0].innerHTML = abc ← 取代內容裡的第1筆內容

document.getElementsByClassName 使用className抓取某些DOM (html標籤)

使用className抓取某些DOM，抓到HTMLCollection物件，可使用陣列方法操作。
  • document.getElementsByClassName(className) ← 抓取HTMLCollection
  • document.getElementsByClassName(className)[0] ← 抓取內容裡的第1筆內容
  • document.getElementsByClassName(className)[1] ← 抓取內容裡的第2筆內容
  • document.getElementsByClassName(className)[0].innerHTML = abc ← 取代內容裡的第1筆內容
